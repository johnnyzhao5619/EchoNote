# Technology Stack

## Build System

- **Vite** - Fast build tool and development server
- **PostCSS** - CSS processing with Autoprefixer
- **TypeScript** - Type checking and compilation

## Tech Stack

- **Programming language**: TypeScript/JavaScript
- **Framework**: Vue 3 (Composition API)
- **Styling**: Tailwind CSS v4 with Typography plugin
- **Internationalization**: Vue I18n v9
- **Routing**: Vue Router v4
- **Testing framework**: TBD (Vitest recommended)
- **Deployment**: GitHub Pages (static site)

## Dependencies

### Core Dependencies

- Vue 3.5.22 - Progressive JavaScript framework
- Vue Router 4.6.3 - Official router for Vue.js
- Vue I18n 9.14.5 - Internationalization plugin

### Development Dependencies

- Vite 7.1.11 - Build tool and dev server
- TypeScript 5.9.0 - Type checking
- Tailwind CSS 4.1.16 - Utility-first CSS framework
- ESLint 9.37.0 - Code linting
- Prettier 3.6.2 - Code formatting

## Common Commands

### Development

```bash
# Start development server
npm run dev
```

### Testing

```bash
# Run type checking
npm run type-check

# Run linting
npm run lint

# Format code
npm run format
```

### Building

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

### Installation

```bash
# Install dependencies
npm install
```

## Code Style

- Follow Vue 3 Composition API patterns
- Use TypeScript for type safety
- Follow ESLint and Prettier configurations
- Use Tailwind CSS utility classes for styling
- Maintain consistent naming conventions (kebab-case for files, PascalCase for components)
- Use semantic HTML and accessibility best practices
