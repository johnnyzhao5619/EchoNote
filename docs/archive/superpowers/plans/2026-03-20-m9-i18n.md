# M9: Internationalization (i18n) Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 实现完整的三语言国际化系统（zh_CN / en_US / fr_FR），支持语言即时切换、持久化、懒加载命名空间及前端轻量 hook，无需第三方 i18n 库。

**Architecture:** Rust 后端在启动时递归加载 `resources/translations/{locale}/{namespace}.json`，合并为内存中的嵌套 `HashMap`，通过 Tauri command 按需返回给前端；前端使用 Zustand store 缓存已加载的命名空间，`useI18n` hook 提供点分路径查找与插值替换，语言切换通过 `set_locale` command 写 `app_settings.locale` 并 emit `locale:changed` 事件触发全局重载。

**Tech Stack:** `serde_json`（JSON 解析）, Zustand（前端状态）, Tauri events（语言变更通知）, shadcn/ui Select（语言选择器）

  ```rust
  //! i18n module: loads translation JSON files at startup and provides lookup.
  //!
  //! Translation files live at:
  //!   resources/translations/{locale}/{namespace}.json
  //!
  //! In-memory structure:
  //!   HashMap<locale, HashMap<namespace, serde_json::Value>>
  //!
  //! Supported locales: en_US, zh_CN, fr_FR
  //! Fallback chain: requested_locale → en_US → key itself (no panic)

  use std::collections::HashMap;
  use std::path::Path;

  use serde::{Deserialize, Serialize};
  use serde_json::Value;
  use specta::Type;

  // ─── Public types ──────────────────────────────────────────────────────────

  /// Metadata about a supported locale, returned to the frontend.
  #[derive(Debug, Clone, Serialize, Deserialize, Type)]
  pub struct LocaleInfo {
      /// BCP-47 style code stored with underscore: "en_US", "zh_CN", "fr_FR"
      pub code: String,
      /// English display name

  impl I18nRegistry {
      /// Recursively walk `translations_dir` and load all
      /// `{locale}/{namespace}.json` files.
      ///
      /// Silently skips files that fail to parse (logs a warning instead).
      pub fn load_from_dir(translations_dir: &Path) -> Self {
          let mut registry = I18nRegistry::default();

          let read_dir = match std::fs::read_dir(translations_dir) {
              Ok(d) => d,
              Err(e) => {
                  tracing::warn!(
                      "i18n: could not read translations directory {:?}: {}",
                      translations_dir,
                      e
                  );
                  return registry;
              }
          };

          for locale_entry in read_dir.flatten() {
              let locale_path = locale_entry.path();
              if !locale_path.is_dir() {
                  continue;
              }
              let locale_code = locale_path
                  .file_name()
                  .and_then(|n| n.to_str())
                  .unwrap_or("")
                  .to_owned();

              let ns_map = registry
                  .data
                  .entry(locale_code.clone())
                  .or_insert_with(HashMap::new);

              let ns_read_dir = match std::fs::read_dir(&locale_path) {
                  Ok(d) => d,
                  Err(e) => {
                      tracing::warn!(
                          "i18n: could not read locale dir {:?}: {}",
                          locale_path,
                          e
                      );
                      continue;
                  }
              };

              for ns_entry in ns_read_dir.flatten() {
                  let ns_path = ns_entry.path();
                  if ns_path.extension().and_then(|e| e.to_str()) != Some("json") {
                      continue;
                  }
                  let namespace = ns_path
                      .file_stem()
                      .and_then(|s| s.to_str())
                      .unwrap_or("")
                      .to_owned();

                  match std::fs::read_to_string(&ns_path) {
                      Ok(contents) => match serde_json::from_str::<Value>(&contents) {
                          Ok(value) => {
                              ns_map.insert(namespace, value);
                          }
                          Err(e) => {
                              tracing::warn!(
                                  "i18n: failed to parse {:?}: {}",
                                  ns_path,
                                  e
                              );
                          }
                      },
                      Err(e) => {
                          tracing::warn!(
                              "i18n: failed to read {:?}: {}",
                              ns_path,
                              e
                          );
                      }
                  }
              }
          }

          tracing::info!(
              "i18n: loaded {} locale(s): {:?}",
              registry.data.len(),
              registry.data.keys().collect::<Vec<_>>()
          );

          registry
      }

      // ─── Public API ──────────────────────────────────────────────────────

      /// List all locales that have at least one namespace loaded.
      pub fn list_available_locales(&self) -> Vec<LocaleInfo> {
          let catalogue = locale_catalogue();
          catalogue
              .into_iter()
              .filter(|info| self.data.contains_key(&info.code))
              .collect()
      }

      /// Return the full parsed JSON Value for a namespace in a locale.
      ///
      /// Fallback chain:
      ///   1. requested locale + namespace
      ///   2. "en_US" + namespace
      ///   3. null (namespace not found anywhere)
      pub fn get_namespace(&self, locale: &str, namespace: &str) -> Option<&Value> {
          self.data
              .get(locale)
              .and_then(|ns_map| ns_map.get(namespace))
              .or_else(|| {
                  self.data
                      .get("en_US")
                      .and_then(|ns_map| ns_map.get(namespace))
              })
      }

      /// Look up a dot-separated key path within a locale+namespace.
      ///
      /// - Supports arbitrary nesting depth: "status.recording"
      /// - Returns the string value if leaf is a JSON string.
      /// - If the key is missing: returns the key path itself (never panics).
      /// - If `args` is provided: replaces `{name}` placeholders.
      ///
      /// Fallback chain:
      ///   1. requested locale → en_US → key path itself
      pub fn translate(
          &self,
          locale: &str,
          namespace: &str,
          key: &str,
          args: Option<&HashMap<String, String>>,
      ) -> String {
          let raw = self
              .lookup_key(locale, namespace, key)
              .or_else(|| self.lookup_key("en_US", namespace, key));

          let text = match raw {
              Some(s) => s,
              None => return key.to_owned(), // graceful fallback: return key itself
          };

          match args {
              Some(vars) if !vars.is_empty() => interpolate(&text, vars),
              _ => text,
          }
      }

      // ─── Internal helpers ─────────────────────────────────────────────────

      fn lookup_key(&self, locale: &str, namespace: &str, key: &str) -> Option<String> {
          let ns_value = self.data.get(locale)?.get(namespace)?;
          resolve_dot_path(ns_value, key)
      }
  }

  // ─── Dot-path resolver ────────────────────────────────────────────────────

  /// Walk a nested `serde_json::Value` using a dot-separated key path.
  /// Returns `Some(String)` only when the final node is a JSON string.
  fn resolve_dot_path(root: &Value, key: &str) -> Option<String> {
      let mut current = root;
      for segment in key.split('.') {
          current = current.get(segment)?;
      }
      current.as_str().map(str::to_owned)
  }

  // ─── Interpolation ────────────────────────────────────────────────────────

  /// Replace `{name}` placeholders in `text` with values from `args`.
  fn interpolate(text: &str, args: &HashMap<String, String>) -> String {
      let mut result = text.to_owned();
      for (k, v) in args {
          result = result.replace(&format!("{{{}}}", k), v);
      }
      result
  }
  ```

  ```rust
  /// Emitting the event allows all open frontend windows to reload their translations
  /// immediately without a restart.

  #[specta::specta]

      locale: String,
      app: AppHandle,
      state: State<'_, AppState>,
  ) -> Result<(), AppError> {

      {
          let mut config = state.config.write().await;
          config.locale = locale.clone();
      }

      // Persist to database
      let now = chrono::Utc::now().timestamp_millis();
      let locale_json = serde_json::to_string(&locale).map_err(|e| AppError::Storage(e.to_string()))?;
      sqlx::query!(
          "INSERT INTO app_settings (key, value, updated_at) VALUES ('locale', ?, ?)
           ON CONFLICT(key) DO UPDATE SET value = excluded.value, updated_at = excluded.updated_at",

      )

      .await
      .map_err(|e| AppError::Storage(e.to_string()))?;

      // Notify all frontend windows so they can reload translations instantly
      app.emit("locale:changed", &locale)
          .map_err(|e| AppError::Io(e.to_string()))?;

      tracing::info!("locale changed to {}", locale);
      Ok(())

  ///
  /// On macOS/Windows the preferred locale is available via the `sys-locale` crate.
  /// Falls back to "en_US" if detection fails or the locale is unsupported.

  #[specta::specta]
  pub async fn get_system_locale() -> Result<String, AppError> {
      let raw = sys_locale::get_locale().unwrap_or_default();
      // Normalise: "zh-CN" → "zh_CN", "fr-FR" → "fr_FR"
      let normalised = raw.replace('-', "_");
      let code = if SUPPORTED_LOCALES.contains(&normalised.as_str()) {
          normalised
      } else {
          // Partial match: try language-only prefix ("zh" → "zh_CN")
          let prefix = normalised.split('_').next().unwrap_or("en");
          SUPPORTED_LOCALES
              .iter()
              .find(|s| s.starts_with(prefix))
              .map(|s| s.to_string())
              .unwrap_or_else(|| "en_US".to_owned())
      };
      Ok(code)
  }
  ```

  ```typescript
  /**
   * i18n Zustand store.
   *
   * Responsibilities:
   *   - Detect initial locale (saved → system → 'en_US')
   *   - Lazily load namespaces via get_translations IPC command
   *   - Expose setLocale() which persists to backend and reloads translations
   *   - Listen to 'locale:changed' Tauri event for multi-window sync
   */

  import { create } from 'zustand'
  import { listen } from '@tauri-apps/api/event'
  import {
    getLocale,
    setLocale as setLocaleCmd,
    getTranslations,
    getSystemLocale,
    listLocales,
  } from '../lib/bindings'
  import type { LocaleInfo } from '../lib/bindings'

  // ─── Types ─────────────────────────────────────────────────────────────────

  type Translations = Record<string, unknown>
  type NamespaceCache = Map<string, Translations>

  export interface I18nStore {
    /** Current active locale code, e.g. "zh_CN" */
    locale: string
    /** Available locales returned by the backend */
    availableLocales: LocaleInfo[]
    /** namespace → parsed translation object */
    namespaces: NamespaceCache
    /** Whether the initial locale has been resolved */
    ready: boolean

    // ── Actions ────────────────────────────────────────────────────────────
    /** Bootstrap: detect saved/system locale and load initial namespaces */
    init: () => Promise<void>
    /** Load a namespace if not already cached (lazy, deduped) */
    loadNamespace: (namespace: string) => Promise<void>
    /** Change locale, persist to backend, reload all cached namespaces */
    setLocale: (locale: string) => Promise<void>
  }

  // ─── Supported locale list (mirrors Rust SUPPORTED_LOCALES) ───────────────

  const SUPPORTED_LOCALES = ['en_US', 'zh_CN', 'fr_FR'] as const
  type SupportedLocale = (typeof SUPPORTED_LOCALES)[number]

  function isSupportedLocale(code: string): code is SupportedLocale {
    return (SUPPORTED_LOCALES as readonly string[]).includes(code)
  }

  // ─── System locale detection ───────────────────────────────────────────────

  /**
   * Detect the best initial locale from:
   *   1. Backend-persisted setting (get_locale IPC)
   *   2. Backend system locale detection (get_system_locale IPC)
   *   3. navigator.language (pure browser fallback, normalise 'zh-CN' → 'zh_CN')
   *   4. Hardcoded 'en_US'
   */
  async function detectInitialLocale(): Promise<string> {
    // 1. Persisted user preference
    try {
      const saved = await getLocale()
      if (saved && isSupportedLocale(saved)) return saved
    } catch {
      // DB not yet initialised on first launch — continue
    }

    // 2. Backend system locale (more reliable than navigator.language on desktop)
    try {
      const system = await getSystemLocale()
      if (system && isSupportedLocale(system)) return system
    } catch {
      // Fall through
    }

    // 3. Browser navigator.language fallback
    const nav = navigator.language.replace('-', '_')
    if (isSupportedLocale(nav)) return nav
    // Partial match: 'zh' → 'zh_CN'
    const prefix = nav.split('_')[0]
    const partial = SUPPORTED_LOCALES.find((l) => l.startsWith(prefix))
    if (partial) return partial

    // 4. Hard fallback
    return 'en_US'
  }

  // ─── Store ─────────────────────────────────────────────────────────────────

  // Track in-flight namespace fetches to deduplicate concurrent requests
  const inflight = new Map<string, Promise<void>>()

  export const useI18nStore = create<I18nStore>((set, get) => ({
    locale: 'en_US',
    availableLocales: [],
    namespaces: new Map(),
    ready: false,

    init: async () => {
      const locale = await detectInitialLocale()

      // Fetch available locales list
      let availableLocales: LocaleInfo[] = []
      try {
        availableLocales = await listLocales()
      } catch {
        // Non-fatal: UI will still work with the static list
      }

      set({ locale, availableLocales })

      // Pre-load the most critical namespaces for immediate UI rendering
      const criticalNamespaces = ['common', 'recording', 'workspace', 'errors']
      await Promise.allSettled(
        criticalNamespaces.map((ns) => get().loadNamespace(ns))
      )

      set({ ready: true })

      // Subscribe to locale:changed events (emitted by Rust set_locale command)
      listen<string>('locale:changed', async (event) => {
        const newLocale = event.payload
        if (!isSupportedLocale(newLocale)) return

        // Clear namespace cache and reload all previously loaded namespaces
        const previousNamespaces = [...get().namespaces.keys()]
        set({ locale: newLocale, namespaces: new Map() })
        inflight.clear()

        await Promise.allSettled(
          previousNamespaces.map((ns) => get().loadNamespace(ns))
        )
      })
    },

    loadNamespace: async (namespace: string) => {
      const { locale, namespaces } = get()

      // Already cached
      if (namespaces.has(namespace)) return

      // Deduplicate concurrent loads for the same namespace
      const key = `${locale}::${namespace}`
      if (inflight.has(key)) {
        return inflight.get(key)!
      }

      const promise = (async () => {
        try {
          const data = await getTranslations(locale, namespace)
          set((state) => {
            const next = new Map(state.namespaces)
            next.set(namespace, data as Translations)
            return { namespaces: next }
          })
        } catch (e) {
          // Non-fatal: key lookups will return the key path itself as fallback
          console.warn(`[i18n] Failed to load namespace "${namespace}" for locale "${locale}":`, e)
        } finally {
          inflight.delete(key)
        }
      })()

      inflight.set(key, promise)
      return promise
    },

    setLocale: async (locale: string) => {
      if (!isSupportedLocale(locale)) {
        console.warn(`[i18n] Unsupported locale: ${locale}`)
        return
      }

      // Persist to backend (will also emit 'locale:changed' event)
      await setLocaleCmd(locale)

      // The locale:changed listener in init() handles the actual state update
      // so we don't double-update here.
    },
  }))
  ```

  ```typescript
  import { useI18nStore } from '../store/i18n'

  // ─── Dot-path resolver ────────────────────────────────────────────────────

  /**
   * Walk a nested object using a dot-separated path.
   * Returns `undefined` if any segment is missing.
   */
  function resolveDotPath(
    obj: Record<string, unknown>,
    path: string
  ): string | undefined {
    const segments = path.split('.')
    let current: unknown = obj
    for (const segment of segments) {
      if (current === null || typeof current !== 'object') return undefined
      current = (current as Record<string, unknown>)[segment]
    }
    return typeof current === 'string' ? current : undefined
  }

  // ─── Interpolation ────────────────────────────────────────────────────────

  /**
   * Replace `{key}` placeholders in `text` with values from `vars`.
   */
  function interpolate(text: string, vars: Record<string, string>): string {
    return Object.entries(vars).reduce(
      (s, [k, v]) => s.replace(`{${k}}`, v),
      text
    )
  }

  // ─── Hook ─────────────────────────────────────────────────────────────────

  export interface UseI18nReturn {
    /** Translate a dot-separated key within the bound namespace */
    t: (key: string, vars?: Record<string, string>) => string
    /** Currently active locale code */
    locale: string
    /** Change locale (persists to backend, triggers global reload) */
    setLocale: (locale: string) => Promise<void>
    /** Whether the i18n system has finished initialising */
    ready: boolean
  }

  /**
   * @param namespace - The translation namespace to bind (e.g. 'recording').
   *                    The namespace is loaded lazily on first call.
   */
  export function useI18n(namespace: string): UseI18nReturn {
    const locale = useI18nStore((s) => s.locale)
    const namespaces = useI18nStore((s) => s.namespaces)
    const ready = useI18nStore((s) => s.ready)
    const loadNamespace = useI18nStore((s) => s.loadNamespace)
    const setLocale = useI18nStore((s) => s.setLocale)

    // Trigger lazy load when this namespace hasn't been loaded yet
    useEffect(() => {
      loadNamespace(namespace)
    }, [namespace, locale, loadNamespace])

    const t = useCallback(
      (key: string, vars?: Record<string, string>): string => {
        const nsData = namespaces.get(namespace) as
          | Record<string, unknown>
          | undefined

        if (!nsData) {
          // Namespace not yet loaded → graceful degradation
          return key
        }

        const raw = resolveDotPath(nsData, key)

        if (raw === undefined) {
          // Key missing → return key path (matches Rust behaviour)
          return key
        }

        return vars ? interpolate(raw, vars) : raw
      },
      [namespace, namespaces]
    )

    return { t, locale, setLocale, ready }
  }

  // ─── Tests (co-located, run with Vitest) ─────────────────────────────────
  //
  // See: src/hooks/__tests__/useI18n.test.ts

  export { resolveDotPath, interpolate }
  ```

## 注意事项

1. **Rust 端错误消息不翻译**：`AppError` 变体的英文消息用于日志，前端根据 error code 从 `errors.json` 显示用户提示。`AppError::Validation` 使用 stable error code（如 `"workspace.duplicate_name"`），前端用点分路径 `t('errors', 'code.workspace.duplicate_name')` 查找。
2. **`sys-locale` crate**：在 macOS 读取 `NSLocale`，在 Windows 读取 `LCID`，在 Linux 读取 `LANG` 环境变量。比 `navigator.language` 更准确，但两者均作备选使用。
3. **翻译文件双写策略**：`resources/translations/` 同时作为开发时源文件和打包资源；通过 Tauri `bundle.resources` glob 嵌入安装包，运行时通过 `resource_dir()` 读取。
4. **命名空间懒加载**：`loadNamespace` 有内存级去重（`inflight` Map），防止同一 namespace 被并发请求多次。已缓存的 namespace 直接从 `namespaces` Map 读取，不发起 IPC。
